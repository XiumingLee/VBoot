<h1 style="text-align: center">VBoot 后台管理系统</h1>

## 项目简介
一个基于 Spring Boot、Spring Security、MyBatis-Plus、Vue的前后端分离的后台管理系统，并且融入了Netty实现了聊天功能。

### 主要功能

- 用户管理
- 角色管理
- 菜单管理
- 权限管理(可以达到`按钮级别`的权限控制)
- 部门管理
- 个人信息管理
- 聊天互动
- 错误日志记录
- 代码生成功能(已删除)
